# Dad Birthday Quest

Это готовый статический сайт-квест для GitHub Pages.

## Что внутри

- `index.html` — весь сайт в одном файле.
- Никакого сервера.
- Никакой базы данных.
- Никакой сборки.
- Работает локально двойным кликом и на GitHub Pages.

## Как изменить вопросы

Открой `index.html` и найди блок:

```js
const QUESTIONS = [...]
```

Там можно менять:
- `title` — заголовок вопроса
- `note` — пояснение
- `answers` — варианты ответа
- `correct` — номер правильного ответа, начиная с 0
- `success` — текст после правильного ответа
- `fail` — текст после неправильного ответа

## Как добавить фото

1. Создай папку `assets` рядом с `index.html`.
2. Положи туда фото, например:
   - `photo1.jpg`
   - `photo2.jpg`
   - `photo3.jpg`
3. В `index.html` найди:

```js
const PHOTOS = [...]
```

И замени `src: ""` на:

```js
src: "assets/photo1.jpg"
```

## Как опубликовать через GitHub Pages

1. Создай новый репозиторий на GitHub, например `dad-birthday`.
2. Загрузи `index.html` в корень репозитория.
3. Открой Settings → Pages.
4. В Build and deployment выбери:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
5. Сохрани.
6. Через пару минут сайт будет доступен по ссылке вида:

```text
https://USERNAME.github.io/dad-birthday/
```
